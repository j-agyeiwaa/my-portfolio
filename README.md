# My Portfolio Website
Welcome to my personal portfolio website. This project is a multi-page responsive website designed to showcase my academic progress, technical skills and software projects.

## Tech Stack
* **HTML5:** Structured semantic markup for multi-page layouts.
* **CSS3:** Custom styling, layout management and high-contrast styling.

## Project Structure
The repository is organized into the following core files and directories:
* 'index.html' - The homepage welcoming visitors with a professional overview.
* 'about.html' - A detailed page outlining my background, education and technical journey.
* 'projects.html' - A gallery showcasing my coding projects and technical work.
* 'blog.html' - A space for sharing insights, updates and documented learning.
* 'contact.html' - A professional contact form and links to my online presence.
* '/images' - Directory containing images.

## How To View Locally
1. Clone or download this repository to your computer.
2. Open the folder in a code editor like VS Code.
3. Open 'index.html' in your preferred web browser (or right-click and use the **Live Server** extension in VS Code) to view the site live.
